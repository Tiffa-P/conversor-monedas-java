Este es un proyecto desarrollado en **Java 17** que permite realizar conversiones entre diferentes monedas internacionales.

### Monedas soportadas:
- 🇰🇷 Won Coreano (KRW)
- 🇺🇸 Dólar Estadounidense (USD)
- 🇨🇴 Peso Colombiano (COP)
- 🇯🇵 Yen Japonés (JPY)
- 🇲🇽 Peso Mexicano (MXN)
- 🇬🇧 Libra Esterlina (GBP)
